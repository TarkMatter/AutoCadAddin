# AutoCadAddin — AutoCAD カスタムコマンド アドイン

AutoCAD上で中心線・下書線をコマンド一発で作成できるC#製アドインです。
図面作成時に繰り返す「線を引く→種類を設定する」という定型操作を自動化します。

---

## 解決する課題

| 課題 | このアドインでの解決策 |
|------|----------------------|
| 中心線を引くたびに線種・レイヤーを手動設定 | `CL` コマンド1つで中心線を即作成 |
| 円の中心線を毎回手動で2本引く | `CCE` コマンドで円を選ぶだけで自動生成 |
| 下書線（補助線）の設定が毎回面倒 | `XL` / `XH` / `XV` で一発作成 |

---

## 使用できるコマンド

| コマンド | 内容 |
|---------|------|
| `CL` | 中心線を作成（2点指定） |
| `CE` | 中心線を作成（エンティティ指定） |
| `CCE` | 円の中心線を作成（円を選択するだけ） |
| `XL` | 下書線を作成 |
| `XH` | 下書線（水平）を作成 |
| `XV` | 下書線（垂直）を作成 |

---

## セットアップ

```bash
# 1. リポジトリをクローン
git clone https://github.com/TarkMatter/AutoCadAddin.git

# 2. Visual Studio 2022 でソリューションを開く
#    AutoCadAddin.sln

# 3. ビルドして AutoCadAddin.dll を生成

# 4. AutoCAD を起動し、コマンドラインに以下を入力
NETLOAD

# 5. ファイル選択ダイアログで AutoCadAddin.dll を指定

# 6. 以降、各コマンドが使用可能になります
```

---

## 注意事項

- AutoCAD フル版が必要です（LT版は .NET API 非対応）
- 本リポジトリは実務で使用したアドインをベースにした公開用サンプルです
- 機密情報・実際の図面名・ブロック名は含まれていません
- AutoCAD のバージョンによってはビルド設定の変更が必要な場合があります
