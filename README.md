# AutoCadAddin

AutoCAD の定型作図を自動化する C# アドイン（.NET API）。
中心線・下書線などの繰り返し作図をコマンド化し、作図の手数を減らします。

> **背景**
> 機械設計の現場では、類似図面の作成や定型的な作図に多くの時間が費やされます。
> 私は実務で、こうした作図を自動化する AutoCAD アドイン（VBA / C# .NET）を開発・運用し、**図面作成時間を最大80%短縮**しました。
> 本リポジトリは、その **基礎となる作図コマンドを抜き出した公開用のサンプル** です。機密情報・実際の図面名・ブロック名・業務固有のロジックは含めていません（実務で運用しているプラグインそのものではありません）。

## 機能（本リポジトリに含まれるコマンド）

AutoCAD のコマンドラインから実行する作図コマンドを登録します。

**中心線系**
- `CL` — 中心線を作成
- `CE` — 中心線を作成
- `CCE` — 円の中心線を作成

**下書線系**
- `XL` — 下書線を作成
- `XH` — 下書線（水平）を作成
- `XV` — 下書線（垂直）を作成

> 実務版では、これらに加えて「類似図面の自動生成」「嵌め合い公差の入力支援」「ブロック内画層の一括変更」などを実装していました（本リポジトリには含めていません）。

## 技術スタック

- **C# / .NET Framework 4.8**
- **AutoCAD 2024 .NET API**（`Autodesk.AutoCAD.Runtime` によるコマンド登録、`Database` / `Editor` 操作）
- **Visual Studio 2022**

AutoCAD の .NET API を用いることで、VBA に比べて保守性・再利用性の高い形でカスタムコマンドを実装できます。

## ビルドと読み込み

1. Visual Studio でビルドし、`AutoCadAddin.dll` を生成
2. AutoCAD で `NETLOAD` コマンドを実行し、生成した `AutoCadAddin.dll` を読み込む
3. コマンドライン（例: `CL`）で各コマンドを実行

## このリポジトリで示していること

- AutoCAD .NET API を用いたカスタムコマンドの設計・登録
- ジオメトリ（線分・円）の生成と、画層・線種の制御
- 現場の繰り返し作業を「コマンド一発」に置き換える、業務自動化の発想
